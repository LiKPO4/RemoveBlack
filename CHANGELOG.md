# Changelog

## v1.5.0

### 修复
- 统一 Windows exe 文件版本号与程序版本号
- 修复打开灰度图时程序崩溃的问题
- 修复启动时状态栏引导提示被覆盖丢失的问题

### 改进
- 多文件拖入时自动进入批量处理
- 批量处理对话框支持选择输出目录与递归子文件夹
- 导出 PNG 时自动补全 `.png` 扩展名
- 记住上次使用的打开/批量目录
- 优化工作线程生命周期，窗口关闭时安全等待线程结束
- 状态栏错误信息改为弹窗提示，避免被忽略
- 关于对话框不再硬编码历史版本说明
- HSV 算法增加参数边界校验
- 工具栏按钮统一创建，缩放百分比显示增加「相对适配」说明
- Release CI 增加测试前置、升级发布 action 到 v2

## v1.4.9

### 改进
- 移除工具栏中的「选择」工具按钮
- 软件启动时默认选中「画笔」工具
- 缩放与平移仍通过滚轮、中键拖动完成，不受工具选择影响

## v1.4.8

### 改进
- 优化底部「打开图片」「导出 PNG」「批量处理」三个按钮的视觉效果
- 统一按钮边框、背景色、悬停与按下状态，避免两侧按钮融入背景
- 导出 PNG 按钮仍保持淡雅绿色，作为主打操作突出显示

## v1.4.7

### 改进
- 「帮助」菜单中的「检查更新」改为常驻显示
- 无新版本时点击「检查更新」，弹窗提示「当前已是最新版本」
- 有新版本时菜单项显示红点图标，并显示「有新版本 vX.X.X」

## v1.4.6

### 新增功能
- **自动更新检查**：软件启动时自动检测 GitHub 最新 release
- **菜单栏更新入口**：发现新版本时，「帮助」菜单显示「🔔 有新版本 vX.X.X」
- **更新下载弹窗**：点击更新入口后弹出对话框，显示当前版本、最新版本及本版更新摘要
- **一键下载新版**：点击「下载新版」即可将最新 `RemoveBlack.exe` 自动下载到当前程序同目录，保存为 `RemoveBlack-v{版本号}.exe`
