# Changelog

## v1.4.8

### 改进
- 优化底部「打开图片」「导出 PNG」「批量处理」三个按钮的视觉效果
- 统一按钮边框、背景色、悬停与按下状态，避免两侧按钮融入背景
- 导出 PNG 按钮仍保持淡雅绿色，作为主打操作突出显示

## v1.4.7

### 改进
- 「帮助」菜单中的「检查更新」改为常驻显示
- 无新版本时点击「检查更新」，弹窗提示「当前已是最新版本」
- 有新版本时菜单项显示红点图标，并显示「有新版本 vX.X.X」

## v1.4.6

### 新增功能
- **自动更新检查**：软件启动时自动检测 GitHub 最新 release
- **菜单栏更新入口**：发现新版本时，「帮助」菜单显示「🔔 有新版本 vX.X.X」
- **更新下载弹窗**：点击更新入口后弹出对话框，显示当前版本、最新版本及本版更新摘要
- **一键下载新版**：点击「下载新版」即可将最新 `RemoveBlack.exe` 自动下载到当前程序同目录，保存为 `RemoveBlack-v{版本号}.exe`

## v1.4.5

### 改进
- 导出 PNG 按钮颜色调整为更淡雅的绿色
- 「打开图片」「导出 PNG」「批量处理」三个按钮尺寸统一

## v1.4.4
